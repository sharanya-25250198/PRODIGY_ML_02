# PRODIGY_ML_02

## Task
Create a K-Means clustering algorithm to group customers of a retail store based on their purchase history.

## Dataset
Mall Customers Dataset

## Technologies Used
- Python
- Pandas
- Matplotlib
- Scikit-learn

## Concepts Used
- K-Means Clustering
- Elbow Method
- Data Visualization

## Output
The project groups customers into different clusters based on:
- Annual Income
- Spending Score
